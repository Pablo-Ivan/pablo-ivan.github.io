Proyectos

Alpha System

Alpha es un sistema neuronal escrito en AIML. Básicamente asimila el lenguage humano y lo convierte en un comando procesable. El lenguage AIML (Artificial Intelligence Markup Language) es una extensión de HTML, mismo en el que la web está escrita. Se le llama lenguage de marcado debido a que todas las intruccionas van entre etiquetas, diferenciándose de un lenguaje íntegro de programación. Su núcleo se basa en una relación categoría>patrón>respuesta, siendo el patrón lo que el usuario escribe. Las respuestas de Alpha se encuentran en inglés por motivos de accesibilidad. He aquí un ejemplo de código en AIML: 

          
<?xml version=“1.0” encoding=“UTF–8”?>
  <aiml version=“2.0”>
    <category>
      <pattern>Hello</pattern>
      <template>Hi, !</template>
    </category>
    <category>
      <pattern>Who are you</pattern>
      <template>I am Alpha</template>
    </category>
  </aiml>
</?xml>

Le explicaré cómo funciona. La primera línea (en dónde se específica la versión de XML y codificación) se conoce como prolog, esta línea le indica al navegador que está ante una versión estandarizada (que cumple con las especificaciones originales) de XML. A continuación viene la etiqueta <aiml>, esta etiqueta está dentro del prolog; en términos de programación a esto se le llama “nesting” y significa que el prolog es la etiqueta padre de la etiqueta aiml. A continuación viene la etiqueta <category>, una categoría es un par pregunta-respuesta, siendo la etiqueta <pattern> lo que el usuario pregunta y <template> lo que el sistema responde.

Pong

Pong, el famoso juego de Atari, lo he rediseñado para que sea compatible con tecnologías de HTML5. La mecánica es la misma, a diferencia de que aquí el oponente es invencible. Escribí el juego con un elemento <canvas>, que es propio de HTML para dibujar un lienzo en la página web. Para que el adversario pueda predecir los movimientos de la pelota, ejecuta un algoritmo geométrico que emplea el método cartesiano para encontrar las coordenadas correctas. Claro que en algún punto puede perder sólo por una cosa: la velocidad de la pelota. En cualquier momento la pelota se moverá más rápido que Alpha y este perderá...si antes no ha perdido usted. Como programador, he vencido a mi juego sólo una vez, y fue por un error en la AI.

Toe

Este clásico juego 'Tic Tac Toe' fue mi primer proyecto que integra HTML, CSS y JavaScript. Al igual que en Pong, el usuario tendrá a un oponente invencible. Su funcionamiento se basa en un algoritmo de teoría de juegos llamado Minimax; se le llama así por la mínima posibilidad de que el jugador gane y la máxima posibilidad de que la computadora gane. Debo confesar que pese a haber programado el juego, nunca he sido capaz de vencerlo ¿, está seguro(a) de poder ganar? Nota: Toe está siendo rediseñado para complementar los estilos del sitio web.

Diplomacy

Diplomacy es un proyecto que surge con la necesidad de un producto para una actividad escolar relacionada a la mercadotecnia. El equipo completo lo constituimos Aldo G. Cabrera Aguilera (Diseño), Erick Mauricio M. Valdéz (Investigación de Mercado), Elioth Fernando R. Rodríguez (Publicidad), Ingrid Y. Bautista Zúñiga (Experiencia de Usuario), (Usuario Final) y, naturalmente, yo Pablo Iván G. Soto (Desarrollo). Está orientada a los estudiantes de secundaria, permitiéndoles practicar las materias que ellos elijan. Al integrarla con Alpha System, la aplicación se convierte en una base de datos colosal, pues el usuario tiene la capacidad de realizar cualquier pregunta relacionada con una materia de estudio. También contará con una segunda versión, que es la "Versión para Profesores", en esta los docentes podrán monitorear las cuentas de los estudiantes y distribuir sus ejercicios.
